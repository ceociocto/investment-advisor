# Logo Design for InvestIQ

## Design Overview

This document presents the modern minimalist logo design for **InvestIQ - AI Investment Strategy Advisor**.

## Design Philosophy

The logo embodies three core principles:

1. **Modernity**: Clean geometric shapes and contemporary typography reflect the AI-powered nature of the platform
2. **Simplicity**: Minimal design ensures instant recognition and versatility across all mediums
3. **Professionalism**: Sophisticated color palette conveys trust, expertise, and financial reliability

## Logo Files

### Main Logo (`logo.svg`)
- **Format**: SVG (vector)
- **Background**: White
- **Elements**: Trending chart with ascending nodes representing investment growth
- **Symbolism**: 
  - Rising trend line = growth and prosperity
  - Golden nodes = value and premium service
  - Connected dots = AI intelligence and data analysis
- **Usage**: Primary brand identifier for web, print, and marketing materials

### Alternative Logo - Hex (`logo-hex.svg`)
- **Format**: SVG (vector)
- **Background**: Deep navy (#0A2463)
- **Elements**: Hexagonal geometric structure
- **Symbolism**:
  - Hexagon = stability and balance
  - Nested shapes = layers of intelligence and depth
  - Dark background = sophistication and premium positioning
- **Usage**: Dark theme applications, premium materials, alternative contexts

### Minimal Logo (`logo-minimal.svg`)
- **Format**: SVG (vector)
- **Background**: White
- **Elements**: Concentric circles with wave pattern
- **Symbolism**:
  - Circular motion = continuous market monitoring
  - Wave pattern = adaptability and intelligence
  - Clean typography = modern fintech aesthetic
- **Usage**: Simplified applications, minimalist design contexts

### Favicon (`favicon.svg`)
- **Format**: SVG (vector)
- **Size**: 100x100px
- **Elements**: Simplified hexagon with center point
- **Usage**: Browser tabs, mobile app icons, small-scale applications

## Color Palette

### Primary Colors
- **Gold**: `#C4A77D` → `#8B7355` (gradient)
  - Represents: Wealth, premium quality, success
- **Navy Blue**: `#0A2463` → `#1e3a8a` (gradient)
  - Represents: Trust, professionalism, stability

### Accent Colors
- **Deep Green**: `#034732` → `#065f46` (gradient)
  - Represents: Growth, financial success
- **White**: `#ffffff`
  - Background for contrast and clarity

## Typography

- **Primary Font**: Arial, sans-serif (web-safe alternative to premium fonts)
- **Logo Text**: Bold, uppercase "INVESTIQ" with letter-spacing
- **Tagline**: Lighter weight, increased letter-spacing for modern feel

## Usage Guidelines

### Minimum Sizes
- **Print**: 25mm (1 inch) width
- **Digital**: 200px width for web
- **Favicon**: 32x32px minimum

### Clear Space
Maintain clear space around the logo equal to the height of the "I" in "InvestIQ" on all sides.

### Backgrounds
- Use on white or light backgrounds (main logo)
- Use on dark backgrounds (hex logo)
- Maintain sufficient contrast for accessibility

### Don'ts
- ❌ Don't stretch or distort the logo
- ❌ Don't change the color proportions
- ❌ Don't add effects (shadows, gradients not in design)
- ❌ Don't rearrange the elements
- ❌ Don't use low-resolution versions

## File Specifications

All logos are provided in SVG format for:
- **Scalability**: Infinite resolution without quality loss
- **Editability**: Can be modified in vector software
- **Web optimization**: Small file size, fast loading
- **Print readiness**: High-quality output at any size

## Implementation

The logos are ready to use in:
1. **Web Application**: Add to Next.js public directory
2. **Marketing Materials**: Use in presentations, brochures
3. **Social Media**: Profile pictures, cover images
4. **Mobile Apps**: App icons, splash screens
